Работа со списками средствами HTML

Маркированные списки

Используется тэг <UL>

<UL> - начало маркированного списка

</UL> - конец маркированного списка

<LI> - элемент списка. Закрывающий тэг не нужен.

Параметры тега <UL>:

Тэг <UL> имеет один параметр – Type, который определяет вид маркировки:

<UL Type=circle> - окружность (◦). Используется по умолчанию.

<UL Type=disc> - черный кружочек (•)

<UL Type=square> - черный квадрат (▪)

Примеры:

Код Web-страницы

Содержимое Web-страницы

<UL Type=square>

<LI>Чебурашка

<LI>Крокодил Гена

<LI>Шапокляк

</UL>
  • Чебурашка
  • Крокодил Гена
  • Шапокляк

 

<UL Type=circle>

<LI>Чебурашка

<LI>Крокодил Гена

<LI>Шапокляк

</UL>
  • Чебурашка
  • Крокодил Гена
  • Шапокляк

 

<UL Type=disc>

<LI>Чебурашка

<LI>Крокодил Гена

<LI>Шапокляк

</UL>
  • Чебурашка
  • Крокодил Гена
  • Шапокляк

 

 

Нумерованные списки

Используется тэг <OL>

<OL> - начало нумерованного списка

</OL> - конец нумерованного списка

<LI> - элемент списка. Закрывающий тэг не нужен.

Параметры тега <OL>:

Тэг <OL> имеет два параметра:

1. Type, который определяет вид нумерации:

<OL Type=1> - арабские цифры ( 1, 2, 3…). Используется по умолчанию.

<OL Type=A> - заглавные латинские буквы (A, B, C…)

<OL Type=a> - строчные латинские буквы (a, b, c…)

<OL Type=I> - заглавные римские цифры (I, II, III…)

<OL Type=i> - строчные римские цифры (i, ii, iii…)

Любое другое значение параметра Type превращает список в нумерованный по умолчанию.

2. Start=число – число (номер элемента), с которого будет начинаться нумерованный список

Пример:

Код Web-страницы

Содержимое Web-страницы

<OL Type=I start=3>

<LI>Чебурашка

<LI>Крокодил Гена

<LI>Шапокляк

</OL>

 III.            Чебурашка

IV.            Крокодил Гена

   V.            Шапокляк

 

Cписки определений

Используется тэг <DL>

<DL> - начало списка

</DL> - конец списка

<DT> - термин. Закрывающий тэг не нужен.

<DD> - определение термина. Закрывающий тэг не нужен.

Параметры: нет

Пример:

Код Web-страницы

Содержимое Web-страницы

<DL>

<DT>FDD

<DD>Дисковод для дискет

<DT>HDD

<DD>Жесткий диск

</DL>

FDD

Дисковод для дискет

HDD

Жесткий диск

 

 

Смешанные (комбинированные) списки

При создании Web-страниц иногда требуется создавать смешанные списки, т.е. сочетать разные виды и стили списков в одном документе.

В этом случае более рациональным является:

–    Описать сначала внешний список

–    Вставить в нужное место внутренний список

Пример:

Код Web-страницы

Содержимое Web-страницы

<OL type=I>

<LI>Устройства вывода

<UL Type=square>

<LI>Монитор

<LI>Принтер

</UL>

<LI>Устройства ввода

<UL Type=square>

<LI>Клавиатура

<LI>Мышь

<LI>Сканер

</UL>

</OL>

       I.            Устройства вывода

                                 §            Монитор

                                 §            Принтер

    II.            Устройства ввода

                                 §            Клавиатура

                                 §            Мышь

                                 §            Сканер
