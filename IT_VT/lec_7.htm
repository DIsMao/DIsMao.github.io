Параметры CSS

 

I. Свойства шрифта (символов)

 

font-family: шрифт – задает шрифт символов

font-style: значение - задает начертание символов

значения:

normal – обычный (по умолчанию)

italic – курсив

oblique – наклонный эффект

font-variant: значение – определяет стиль символов

значения:

normal – обычный шрифт (по умолчанию)

small-caps – маленькие заглавные буквы

font-weight: значение – определяет толщину символов

значения:

normal – обычная толщина (по умолчанию)

lighter – светлое начертание

bold – полужирное начертание

bolder – жирное начертание

100 - 900 – числовой параметр (100 – самый светлый, 900 – самый жирный)

font-size: значение – задает размер символов

Значения:

Normal - обычный (по умолчанию)

Число Pt – размер в пунктах

Число Px – размер в пикселях

Число % - в % от исходного

II. Свойства текста

line-height: значение – задает междустрочный интервал (интерлиньяж)

Значения:

normal – обычный (по умолчанию)

число – множитель (1.5 – полуторный)

число Px – в пикселях

число % - в % от исходного (150% - полуторный)

text-decoration: значение – отвечает за вид линий у текста

Значения:

None – убрать подчеркивание

Underline – включить подчеркивание

Overline – линия над текстом

Line-through - перечеркивание

text-transform: значение – отвечает за работу со строчными и прописными буквами

Значения:

None – обычный (по умолчанию)

Capitalize – капитализация, титульный эффект (каждое слово начинается с заглавной буквы)

Uppercase – все заглавные

Lowercase – все строчные

text-align: значение – определяет выравнивание текста

Значения:

Left – по левому краю

Right – по правому краю

Center – по центру 

Justify – по ширине

text-indent: значение – задает отступ первой (красной) строки

Значения:

Число Px – точное значение в пикселях

Число % - в % от ширины строки

word-spacing: значение – задает расстояние между словами

Значения:

Число Px – точное значение в пикселях

Число Pt - точное значение в типографских пунктах

III. Цвета и рисунки

Color: значение – задает цвет текста

Способы задания цветов:

1.     Английское название (Red, Green ..)

2.     Шестнадцатиричеый код (#0000FF)

3.     Палитра RGB (RGB (0, 0, 255))

Background-color: значение– задает цвет фона

Background-image: URL (путь\файл) – задает фоновый рисунок

Background-repeat: значение – задает повторяемость фонового рисунка

Значения:

repeat – повторять по вертикали и горизонтали (по умолчанию)

repeat-x – повторять только по горизонтали

repeat-y - повторять только по вертикали

no-repeat – отключить повторение

background-attachment: значение – задает прокручиваемость фона вместе с содержимым

Значения:

scroll – включено (по умолчанию)

fixed - выключено

background-position: значение – задает начальное положение фонового рисунка на Web-странице

Значения:

Top – по верхнему краю

Center – по середине

Bottom – по нижнему краю

Left – по левому краю

Right – по правому краю

IV. Гиперссылки

 

A:link {…} - Определяет стиль для обычной непосещенной ссылки

A:visited {…} - Определяет стиль для посещенной ссылки

A:active {…} - Определяет стиль для активной ссылки. Активной ссылка становится при нажатии на нее

A:hover {…} - Определяет стиль для ссылки при наведении на нее мышкой

V. Списки

List-style: значение - Определяет вид списка

Значения:

Disc - круг

Circle - окружность

Square – закрашенный квадрат

Decimal - квадрат

lower-roman – маленькие римские

upper-roman – заглавные римские

lower-alpha – маленькие латинские

upper-alpha – заглавные латинские

none – отключить список

List-style-image: URL (путь\имя) – устанавливает в виде маркера произвольную картинку

List-style-position: значение – определяет положение маркера относительно абзаца с текстом

Значения:

Inside – внутри абзаца

Outside – вне абзаца

VI. Указатели мышки

Cursor: значение – определяет внешний вид указателя мышки

Значения:

Default – стандартный (по умолчанию)

Crosshair - крестик

Move – перекрестие со стрелками

Pointer (Hand) - рука

Progress (Help) – стандартный указатель со знаком вопроса

Text – текстовый указатель

Wait – песочные часы

n-resize – стрелка на север

ne-resize – стрелка на северо-восток

e-resize – стрелка на восток

se-resize – стрелка на юго-восток

s-resize – стрелка на юг

sw-resize – стрелка на юго-запад

w-resize – стрелка на запад

nw-resize – стрелка на северо-запад, стандартная (по умолчанию)

VII. Границы, рамки, отступы

Отступы:

Padding: значение – устанавливает отступ от всех границ элемента до его содержимого

Значения:

Число Px – в пикселях

Число % - в % от размера элемента

Padding-top: значение – устанавливает отступ сверху от границы элемента до его содержимого

Padding-right: значение – устанавливает отступ справа от границы элемента до его содержимого

Padding-bottom: значение – устанавливает отступ снизу от границы элемента до его содержимого

Padding-left: значение – устанавливает отступ слева от границы элемента до его содержимого

 

Границы и рамки

border-style: значение – устанавливает внешний вид рамки

Значения:

None – без рамки

Dotted – мелкий пунктир (точками)

Dashed – крупный пунктир (линиями)

Solid – обычная рамка (по умолчанию)

Double – двойная рамка

Groove – приподнятый эффект рамки

Ridge – утопленный эффект рамки

Inset – эффект рамки «Нажатая кнопка»

Outset – эффект рамки «Ненажатая кнопка»

border-color: цвет – устанавливает цвет рамки

border-width: значение – устанавливает толщину рамки

Значения:

Число Px – в пикселях (по умолчанию: 0)

Thin – тонкая

Medium – средняя

Thick - толстая

border-top-width: значение – устанавливает толщину верхней границы рамки

border-right-width: значение – устанавливает толщину правой границы рамки

border-bottom-width: значение – устанавливает толщину нижней границы рамки

border-left-width: значение – устанавливает толщину левой границы рамки

Линейка прокрутки:

Параметры цветов линейки прокрутки описываются только для тега BODY

SCROLLBAR-FACE-COLOR: цвет – общий цвет линейки прокрутки
SCROLLBAR-HIGHLIGHT-COLOR: цвет – подсветка
SCROLLBAR-SHADOW-COLOR цвет – тень
SCROLLBAR-3DLIGHT-COLOR: цвет – 3-D объекты
SCROLLBAR-ARROW-COLOR: цвет – кнопки со стрелками
SCROLLBAR-TRACK-COLOR цвет – трэк (полоса, по которой движется лифт)
SCROLLBAR-DARKSHADOW-COLOR: цвет – тень 3D-объектов
