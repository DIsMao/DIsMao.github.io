Фреймы

I. Основные понятия

Фреймы (frame — кадр, рамка)  – способ верстки Web-страниц, при котором окно браузера делится на части, в каждую из которых можно загрузить отдельную Web-страницу.

Пример: общая (классическая) структура Web-страницы с фреймами

Заголовок сайта

Верхнее горизонтальное меню (набор ссылок)

Левое вертикальное меню

(набор ссылок)

Область контента

(загружаемая информация)

Правое вертикальное меню

(набор ссылок)

Нижнее горизонтальное меню (набор ссылок)

 

Достоинства фреймов

1. Простота создания

Для создания сайта с использованием фреймов достаточно описать только стартовую страницу (т.е. деление окна на фреймы) и загрузить во фреймы нужные Web-страницы.

2. Простота обновления

Для обновления информации на сайте с фреймами достаточно изменить содержимое контента или добавить новые ссылки.

3. Быстрота загрузки информации

Загрузка Web-страницы происходит только в указанный фрейм, остальные остаются неизменными. Так как при загрузке обновляется только какая-то часть Web-страницы, то время общей загрузки сайта уменьшается

4. Удобство размещения элементов управления (навигации)

Фреймы со ссылками можно размещать точно в нужном месте окна браузера

5. Удобный и интуитивно-понятный механизм навигации

При работе с фреймами используется циклическая навигация, которая позволяет переходить с любой страницы сайта на любую другую

6. Изменение размеров фреймов

Можно изменять размеры фреймов (областей) путем перетаскивания их границ.

Недостатки фреймов

1. Плохая индексация поисковыми системами.

Поисковые системы индексируют только стартовую страницу.

2. Внутренние страницы нельзя добавить в "Избранное".

3. Параметры фреймов могут по-разному интерпретироваться разными браузерами.

4. При выполнении действия «Обновить страницу» всегда открывается стартовая страница.  

Применение фреймов

Фреймы в основном применяются для создания Web-справочников и Web-учебников

II. Создание фреймов

Для создания фрейма применяется парный тег <FRAMESET>, который заменяет тег <BODY> в документе и используется для разделения экрана на области.

Параметры тега <FRAMESET>:

Cols=”…, …, …” – определяет деление страницы на фреймы по горизонтали. Значение может быть абсолютным (в пикселах), либо относительным (в процентах)

Rows=”…, …, …” – определяет деление страницы на фреймы по вертикали

При сложной структуре фрейма сначала описывается вертикальное деление, затем горизонтальное.

 Внутри тега <FRAMESET> находятся одиночные теги <FRAME>, которые указывают на Web-страницы, предназначенные для загрузки в указанные области.

Загружаемые страницы описываются последовательно: сначала сверху - вниз, затем слева – направо (как в таблицах).

Параметры тега <FRAME>:

src="путь/имя" – определяет загружаемую во фрейм Web-страницу

name="имя" – внутреннее имя фрейма (используется в гиперссылках)

 

Примеры описания страниц с фреймами:

1 структура:

Заголовок (Zag)

Меню (Menu)

Область контента (Content)

 

В область заголовка нужно загрузить страницу Zag.html

В панель меню нужно загрузить страницу Menu.html

В область контента нужно загрузить стартовую контентную страницу Content.html (может быть пустой)

Код страницы:

<frameset rows="10%,90%"> - вертикальное деление                          

<frame src="zag.html" name="Zag"> - загрузка заголовка

<frameset cols="20%,80%"> - горизонтальное деление

<frame src="menu.html" name="MENU"> - загрузка меню (ссылок)

<frame src="content.html" name="CONTENT"> - загрузка стартовой контентной страницы (может быть пустой)

</frameset> - конец горизонтального деления

</frameset>- конец вертикального деления

 

2 структура:

Заголовок сайта (Zag)

Верхнее меню (Menu1)

Левое меню (Menu2)

Область контента

(Content)

Правое меню (Menu3)

Нижнее меню (Menu4)

 

<frameset rows="10%,10%,70%,10%"> - вертикальное деление на 4 части в указанном процентном соотношении                       

<frame src="zag.html" name="Zag"> - загрузка заголовка

<frame src="menu1.html" name="Menu1"> - загрузка верхнего меню

<frameset cols="10%,80%,10%"> - горизонтальное деление на 3 части в указанном процентном соотношении

<frame src="menu2.html" name="Menu2"> - загрузка левого меню

<frame src="content.html" name="Content"> - загрузка стартовой контентной страницы

<frame src="menu3.html" name="Menu3"> - загрузка правого меню

</frameset> - конец горизонтального деления

<frame src="menu4.html" name="Menu4"> - загрузка нижнего меню

</frameset>- конец вертикального деления

 

III. Редактирование фреймов

Редактирование кода отдельных областей осуществляется через правую клавишу мышки.

Редактирование структуры фреймов (стартовой страницы с фреймами) осуществляется через меню Вид.

IV. Настройка дополнительных параметров фреймов

Для тега <Frameset> можно применить 2 дополнительных параметра:

Frameborder=число – задает толщину рамок между фреймами в пикселях. Чтобы отключить рамки необходимо поставить значение 0 или NO (можно использовать оба значения для работы с разными браузерами)

Bordercolor=цвет – определяет цвет рамки между фреймами (по умолчанию – серый)

 

Для тега <Frame> можно также применить 2 дополнительных параметра:

Noresize – включает запрет на изменение размеров фрейма

scrolling=no – отключает полосы прокрутки во фрейме

V. Ссылки во фреймах

Чтобы открыть гиперссылку в указанном фрейме, используется команда:

<A href=путь target=имя>Объект ссылки</a>

Где имя – это внутреннее имя фрейма, в котором хотим открыть гиперссылку (напомним, что внутреннее имя записывается в теге <Frame>.
