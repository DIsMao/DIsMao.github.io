Задание фонового рисунка

Описывается внутри тэга <BODY>. Используются параметры:

1. <body background=путь к рисунку> - определяет изображение, которое будет использоваться в качестве фонового рисунка.

В качестве пути к рисунку можно использовать абсолютный или относительный адрес рисунка.

Абсолютный адрес – полный путь к файлу (D:\Images\fon.jpg). Лучше не использовать.

Относительный адрес – путь к файлу с рисунком относительно местоположения Web-страницы.

Если Web-страница и рисунок располагаются рядом (в одной папке), тогда в качестве пути к рисунку достаточно указать только название рисунка.

Если рисунок располагается внутри папки рядом с Web-страницей, тогда в качестве пути к рисунку пишут название папки\название файла с рисунком. Для перехода на один уровень вверх используется две точки (..)

Все Web-проекты рекомендуют располагать в отдельной папке, а для хранения рисунков использовать в этой папке специальную папку (RIS или Images). Имена рисункам лучше давать английские. При описании пути к рисунку обязательно записывать расширение графического файла.

Для фона не устанавливаются ширина и высота, и он всегда отображается в натуральную величину с масштабом 100%. Если рисунок по размеру меньше окна браузера, то картинка повторяется по горизонтали вправо и вниз, выстраиваясь, как мозаика. По этой причине на месте стыка фоновых картинок могут возникнуть видимые перепады, заметные для посетителей сайта.

В качестве фона допускается использовать анимированные изображения в формате GIF, но они отвлекают внимание читателей.

Пример:

На рабочем столе имеется папка Web. В ней имеется файл Primer.html и папка Ris, содержащая рисунок Fon.jpg. Задать в качестве фона рисунок  Fon.jpg (использовать относительную адресацию).

Ответ:

<html>

<body background=Ris\fon.jpg>

</body>

</html>

2. <body bgproperties=fixed> - фиксирует фон на странице.

Если этот параметр не писать, то фоновый рисунок будет прокручиваться вместе с содержимым Web-страницы.

3. <body leftmargin=число> - определяет отступ от левого и правого края окна браузера до контента веб-страницы в пикселах. Значение по умолчанию: 10 пикселов

4. <body scroll=no> - запрещает показ полосы прокрутки в браузере. Если этот параметр не писать, то полосы прокрутки будут отображаться в браузере.

Пример:

<html>

<body background=Ris\fon.jpg bgproperties=fixed leftmargin=20 scroll=no >

</body>

</html>

Ответ:

Web-страница будет иметь фоновый рисунок Fon.jpg из папки Ris

Фоновый рисунок будет зафиксирован на Web-странице

Отступ от левого и правого края окна браузера до содержимого веб-страницы составит 20 пикселов

Полосы прокрутки в браузере будут отключены

Рисование горизонтальной линии

Используется тэг <HR>.

Используются параметры:

1. Align=left, center, right – задает способ выравнивания линии на странице (по левому краю, по центру, по правому краю).

Значение по умолчанию: center

2. Color=цвет – задает цвет линии.

3. Size=число – устанавливает толщину линии в пикселах. Значение по умолчанию: 2.

4. Width=число - устанавливает длину линии в пикселах или в % от ширины окна браузера. Значение по умолчанию: 100%. Лучше использовать относительную размерность, т.е. %.

Пример:

<HR Align=left Width=50% Size=5 Color=Red>

Данная команда описывает линию слева, длина - 50% от окна браузера, толщина – 5 пикселей, цвет – красный

 

Бегущая строка

Используется тэг <Marquee>текст </Marquee> - заставляет текст двигаться в строку справа налево

Параметры:

1. Height=число – высота контейнера в пикселах или в %

2. Width=число  - ширина контейнера в пикселах или в %

3. BGСOLOR=цвет – цвет фона контейнера

4. HSpace=число – отступ от контейнера до текста по горизонтали в пикселах

5. VSpace=число – отступ от контейнера до текста по вертикали в пикселах

6. Scrolldelay=число – задает время задержки между движениями в миллисекундах

7. DIRECTION – способ движения текста

Значения:

DIRECTION=Down - Движение сверху вниз.

DIRECTION=Left - Движение справа налево (по умолчанию).

DIRECTION=Right - Движение слева направо

DIRECTION=Up - Движение вверх.

Работа с рисунками

Используется тэг <IMG>

<IMG> предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG.

Параметры:

1. SRC=путь – определяет абсолютный или относительный адрес графического файла.

(Если вместо SRC написать dynsrc, можно разместить вместо картинки видеоролик, но работает только в IE)

2. Align= - определяет способ выравнивания рисунка, а также способ обтекания рисунка текстом

Значения ALIGN:

Основные:

align=left  - Выравнивает изображение по левому краю окна. Текст обтекает рисунок справа.

align=right - Выравнивает изображение по правому краю окна. Текст обтекает рисунок слева.

Дополнительные:

align=absmiddle  - Выравнивание середины изображения по средине текущей строки

align=baseline - Выравнивание изображения по базовой линии текущей строки

align=bottom - Выравнивание нижней границы изображения по окружающему тексту

align=middle - Выравнивание середины изображения по базовой линии текущей строки.

align=texttop - Верхняя граница изображения выравнивается по самому высокому текстовому элементу текущей строки

align=top - Верхняя граница изображения выравнивается по самому высокому элементу текущей строки.

Значение по умолчанию: bottom

3. Alt=”текст” – задает альтернативный текст для рисунка, т.е. текст, который будет отображаться при невозможности загрузки рисунка в браузер, а в Internet Explorer еще и текст,  который будет отображаться при наведении указателя мышки на рисунок (всплывающая подсказка). Кавычки использовать обязательно. В других браузерах для всплывающей подсказки используется параметр Title.

Рекомендуется использовать оба параметра для всех рисунков и всех браузеров!

4. Border=число – задает ширину рамки вокруг рисунка в пикселах. Значение по умолчанию 0.

5. Height=число – высота рисунка в пикселах или в %

6. Width=число  - ширина рисунка в пикселах или в %

Для параметров Height и Width значениями по умолчанию являются исходные высота и ширина рисунка, которые лучше не менять. Для пропорционального изменения размеров рисунка используется только параметр Width.

7. HSpace=число – отступ от рисунка до текста по горизонтали в пикселах

8. VSpace=число – отступ от рисунка до текста по вертикали в пикселах

Значения по умолчанию: 0

Пример:

Web-страница:

Код Web-страницы:

<html>

<body>

<img src=images\1.gif align=left hspace=20 border=3 alt=«Рыцари креста»>

<h1>Рыцари креста</H1>

<h3>Жанр: стратегия в реальном времени</h3>

<h4>Разработчик: FreeMind SoftWare</h4>

<h4>Увлекательная стратегическая игра, построенная исключительно на реальных исторических фактах</h4>

</body>

</html>
