Лекция №6. Основы CSS

I. Основные понятия

CSS (Cascading Style Sheets) - Каскадные Таблицы Стилей.

CSS - технология описания внешнего вида документа, написанного языком разметки.

Обычный HTML позволяет изменять текст только с помощью тегов начертания, форматирования и использования заголовков. Если понадобится изменить оформление однотипных элементов на сайте, придется просматривать код всех страниц, чтобы найти и поменять параметры форматирования.

Каскадные таблицы стилей (CSS) позволяют хранить параметры форматирования в стилях.

Стиль - набор параметров форматирования, который применяется к элементам Web-страницы.

II. Преимущества использования CSS

1.            Комплексное форматирование. Стили позволяют одним действием применить сразу всю группу атрибутов форматирования для отдельного элемента Web-страницы.

2.            Быстрое изменение стиля. Если требуется быстро изменить внешний вид текста, созданного с помощью одного из стилей, достаточно изменить только параметры этого стиля.

3.            Дополнительные возможности форматирования. По отношению к HTML стили предлагают намного больше возможностей для форматирования.

4.            Быстрая загрузка сайтов. Стили могут храниться во внешнем файле, браузер кэширует такие документы, поэтому загрузка сайта будет происходить быстрее.

5.             Единый стиль оформления. Один файл со стилями можно использовать на множестве Web-страниц.

6.      Создание новых тэгов. Средствами СSS можно создавать новые теги

III. Добавление CSS на Web-страницу

1 способ. Таблицы связанных стилей (Linked Style Sheet)

Самый мощный и удобный способ определения стилей и правил для сайта.

Стили хранятся в отдельном файле c расширением .css, который может быть использован для любых Web-страниц.

Для подключения таблицы связанных стилей к Web-странице используется тег LINK в заголовке страницы.

Подключение таблицы связанных стилей:

<html>

<head>

<link rel="stylesheet" type="text/css" href=файл со стилем>

</head>

В качестве файла со стилем используется относительный путь к Вашему файлу с расширением .css

В файле со стилем используется следующий синтаксис:

тег 1 {параметр1: значение; параметр2: значение; …}

тег 2 {параметр1: значение; параметр2: значение; …}

…..

 

2 способ. Таблицы глобальных стилей (Global Style Sheet)

Стиль хранится на самой Web-странице и распространяется только в пределах текущей Web-страницы.

По своей гибкости и возможностям этот способ использования стиля уступает предыдущему, но также позволяет размещать все стили в одном месте.

Стили описываются в заголовке Web-страницы.

Определение стилей задается тегом STYLE.

Использование таблицы глобальных стилей:

<html>

<head>

 

<style type="text/css">

тег 1 {параметр1: значение; параметр2: значение; …}

тег 2 {параметр1: значение; параметр2: значение; …}

…..

</style>

 

</head>

3 способ. Внутренние стили (Inline Style)

Внутренний стиль применяется для одиночного тега, используемого на Web-странице. Для определения стиля используется параметр style, а его атрибуты указываются с помощью языка стилей.

Использование внутренних стилей:

 

<Тэг style="параметр1: значение; параметр2: значение; …">Объект на странице</Тэг>

4 способ. Смешанные стили

На Web-странице можно использовать любое сочетание стилей. Приоритет в отображении у внутренних стилей, затем идут глобальные стили, после связанные таблицы.

Для Web-сайтов рекомендуется использовать связанные таблицы стилей (файлы с расширением .css)

Для самостоятельных Web-страниц рекомендуется использовать глобальный стиль.
