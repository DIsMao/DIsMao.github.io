Создание Web-страниц кодом HTML в OC Windows

Общий порядок создания Web-страниц

1.      Создать Текстовый документ. Дать ему название. Изменить расширение на html

2.      Открыть созданный файл (по умолчанию файл должен открыться в Internet Explorer)

3.      Выбрать в меню Вид команду Просмотр HTML-кода (по умолчанию редактором HTML является программа Блокнот).

4.      В окне Блокнота набрать код Web-страницы.

5.      Сохранить изменения в коде.

6.      Обновить Web-страницу (команда Вид-Обновить).

  Тэги и дескрипторы

Web-страницы создаются с помощью специальных команд HTML.

Эти команды называются Тэгами или Дескрипторами.

Тэги заставляют текст изменять свой внешний вид в окне обозревателя.

Все тэги HTML заключаются между знаками < и >

Большинство тегов являются парными:

·               открывающий тег: <тег>

·               закрывающий тег: </тег>  

Большинство тэгов могут иметь параметры, которые записываются внутри открывающего тега после названия через пробел.

Параметр должен иметь значение, которое записывается через =

Таким образом, общая структура тега выглядит следующим образом:

<тег параметр1=значение параметр2=значение …> текст </тег>

 Структура кода Web-страницы

<HTML>

<HEAD>

<TITLE>Заголовок Web-страницы</TITLE>

</HEAD>

<BODY>

Содержимое Web-страницы

</BODY>

</HTML>

 

Описание:

<HTML> - Начало страницы

<HEAD> - Начало области заголовка

<TITLE>Заголовок Web-страницы</TITLE>

</HEAD> - Конец области заголовка

<BODY> - Начало тела страницы

Содержимое Web-страницы

</BODY> - Конец тела страницы

</HTML> - Конец страницы

Внутри тегов <HEAD> располагается область заголовка. Она описывает название страницы и может содержать дополнительные команды HTML

У простых Web-страниц область заголовка может отсутствовать

Внутри тегов <TITLE> располагается Заголовок Web-страницы (текст, который будет отображаться в области заголовка программы-браузера).

Внутри тегов <BODY> располагается тело Web-страницы, т.е. все её содержимое

Параметры тега  <BODY>

BGCOLOR=цвет – задает цвет фона Web-страницы

TEXT=цвет – задает основной цвет символов для всей Web-страницы

Способы задания цвета:

1.       Английское название цвета: (Red - красный, Green - зеленый, Blue - синий, Brown-коричневый, Yellow - желтый, Gray – серый и т.д. смотри ТАБЛИЦА ЦВЕТОВ). Также можно использовать слова light  и Dark (пример: lightred – розовый цвет)

2.       16-чный код: три числа от 00 до ff для каждого из трех цветов: красного(R), зеленого(G) и синего(B). 00 – отсутствие цвета, ff – максимум цвета. Пример: FF0000 – красный, 00 FF 00 – зеленый, 0000FF – синий.

Пример: команда <BODY TEXT=0000FF BGCOLOR=YELLOW> задает синий текст на желтом фоне

 

Основы форматирования содержимого Web-страниц

Абзацы

<P> - начало абзаца

</P> - конец абзаца

Абзац – фрагмент текста, который отделяется от окружающего текста пустым пространством.

Тэг <BR> - принудительный переход в новую строку. Тэг <BR> является одиночным.

 

Выравнивание абзаца

Выравнивание абзаца задается с помощью параметра Align

Виды:

<P align=left> текст абзаца</P>  - абзац по левому краю

<P align=center> текст абзаца</P>  - абзац по центру

<P align=right> текст абзаца</P>  - абзац по правому краю

По умолчанию текст выравнивается всегда по левому краю, значит align=left можно не писать.

 

Заголовки

Заголовок – это абзац с заранее определенным стилем форматирования.

HTML поддерживает 6 уровней заголовков:

<H1>текст</H1> - самый крупный

<H2>текст</H2>

<H3>текст</H3>

<H4>текст</H4>

<H5>текст</H5>

<H6>текст</H6> - самый маленький

К заголовкам также можно применять выравнивание (параметр Align)

 

Простейшее форматирование текста

<B>текст</B> - полужирный

<I>текст</I> - курсив

<U>текст</U> - подчеркнутый

<S>текст</S> - перечеркнутый

<TT>текст</TT> - машинописный

<SUB>текст</SUB> - нижний индекс

<SUP>текст</SUP> - верхний индекс

Вышеперечисленные тэги можно использовать в сочетании друг с другом

Пример:

<B><I><U>Интернет</U></I></B> - слово Интернет будет отображено полужирным подчеркнутым курсивом.  

 

Форматирование элементов текста с помощью тега <FONT>

<FONT face=шрифт size=размер color=цвет>Текст</FONT>

face=шрифт – задает шрифт текста (Times New Roman, Arial и т.д.). Длинные шрифты необходимо записывать в ‘ ’.

size=размер – задает размер символов (число от 1 до 7: 1 – самый маленький, 7 - самый большой)

color=цвет – задает цвет текста внутри тегов FONT

Параметры face, size, color можно использовать по отдельности

 

Пример:

Web-страница имеет код:

<HTML>

<HEAD>

<TITLE>Моя первая Web-страница</TITLE>

</HEAD>

<BODY bgcolor=yellow text=blue>

<i><p align=right><FONT face=Times New Roman size=7 color=red>Пример 1</FONT></p></i>

<H1 align=center>Пример2</H1>

</BODY>

</HTML>
